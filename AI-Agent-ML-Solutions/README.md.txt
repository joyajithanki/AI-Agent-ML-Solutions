# AI-Powered Sales Data Analyst

## Description
An AI-powered system that analyzes sales datasets, generates business insights, answers user queries, and provides predictive analytics.

## Features
- CSV Upload
- Automated Data Analysis
- Sales Insights Generation
- Query Answering
- Sales Forecasting
- Interactive Dashboard

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Streamlit

## Duration
60 Days
